# merkle, crypto-js 설치

```sh
npm init -y
npm install -D typescript ts-node
npm install -D @types/merkle merkle
npm install -D @types/crypto-js crypto-js
npm install -D tsc-alias tsconfig-paths
```

```sh
npx tsc --init
```

```sh
npm install -D @types/jest jest
npm install -D ts-jest
```
